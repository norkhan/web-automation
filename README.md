# web-automation
Selenium Webdriver 

### OS
```
Ubuntu 18.04.4 LTS
```

### Environments Setup

| Name  | Version |
| ------------- | ------------- |
| node  | v8.10.0  |

### Installing

Run 'start.sh' script that will install all dependencies and packages and
start web-automation

```
chmod +x start.sh && ./start.sh
```

### Runnig the tests

npm test
